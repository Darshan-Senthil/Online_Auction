# Online_Auction
The Online Auction Website allows users to participate in online auctions. The website is built using JavaScript, Java, SQL Workbench, and Tomcat Server. The front-end interface is developed using HTML, CSS, and JavaScript, while the back-end functionality is implemented using Java and SQL Workbench. The Tomcat Server is used to host the website and handle user requests.

The website offers various features like bidding, listing of items, user authentication, payment processing, and auction management. The project also includes database schema and scripts for creating tables and relationships.

The website is designed to be user-friendly, intuitive, and responsive, with a simple and clean interface. The project includes comprehensive documentation and instructions on how to set up and use the website, making it easy for other developers to contribute to and improve the project
